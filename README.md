# Instructions

1. Install Python 3+ (I tested it in Python 3.13.11)
2. [Create a virtual environment and install requirements.txt](https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/)
3. Activate the virtual environment
4. Run the `microtiter_gui.py`
